## 1. Landing & Markets (`/`)
* **Purpose:** The entry point of the exchange, designed for user acquisition and market overview.
* **Components:** * A real-time data table listing all supported trading pairs (e.g., `BTC_USDT`, `ETH_USDT`).
  * Displays 24h volume, current price (Ticker), and 24h price change percentage.
  * Direct navigation links to the Trading Terminal for each specific pair.n
